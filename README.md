The directory Equitable_Complete_Graphs contains examples of equitable (s,p) edge colorings of K_n. The directory Multiple_Colorings_For_Equitable_Complete_Graphs contains more examples and specifically multiple different colorings for different equitable (s,p) edge colorings of K_n. The directory Smaller_Embedded_Complete_Graphs contains examples of embedded equitable (s,p) edge colorings of smaller K_n that exist in some of the example graphs generated in the Equitable_Complete_Graphs directory. Also included is the code used to generate these directories for both a computer that does not have gpus and a computer that has access to gpus and has high parrelization capability. Important to note that without high computational resources this particular code will quickly run into a ceiling of generating these examples.
